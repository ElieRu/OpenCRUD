Installation

1. Be sure to created the database in mySQL.

2. Initialise your database informations in the `/database/db-infos.php` file.
But by default, you need to initialise the `$dbname` only.

3. Link the `init.php` in the file that you need to instantiate OpenCRUD.

4. Follow the documentation on :
   * https://open-crud-plateform.netlify.app/
   * https://github.com/ElieRu/OpenCRUD
